# 🎓 Student Management System (Java)

## 📌 Overview

This is a simple **console-based Student Management System** built in Java.
It allows users to manage student records with features like adding, viewing, updating, deleting, and searching students.

---

## 🚀 Features

* Add new student records
* View all students in formatted table
* Update existing student details
* Delete student records
* Search by Student ID or Name
* Input validation and error handling

---

## 🛠️ Technologies Used

* Java (Core Java, OOP concepts)
* ArrayList for data storage
* Scanner for user input

---

## 📂 Project Structure

```
src/        → Source code
docs/       → Documentation
test_data/  → Sample data for testing
```

---

## ▶️ How to Run

1. Clone the repository:


   git clone https://github.com/your-username/student-management-system.git
   

2. Navigate to project folder:

   
   cd student-management-system
   

3. Compile:

   
   javac src/*.java
   

4. Run:


   java src.StudentManagementSystem
